# Admin Dashboard

Administrative interface for monitoring service health, viewing metrics, and controlling service lifecycle.

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Axios
- Recharts

## Setup

1. `npm install`
2. `cp .env.example .env` (and configure variables)
3. `npm run dev`

## URLs

- Local: <http://localhost:3001>
